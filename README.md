# CalendarService
## Description
Allows head coaches to create, edit, and delete an event using Google's calendar API. The head coaches can schedule games with the feature.

## Functionality
* Full functionality of Google's calendar API
* Create an event
* Edit an event
* Delete an event
* Create a description for the event
* Select a location for the event
* Add a start time for the event
* Add an end time for the event

## Getting Started
1. Clone this repo to your local machine:
```bash
  git clone https://github.com/OutOfTheParkAtRevature/CalendarService.git
```
2. To create and connect first branch to remote repository branch:
```bash
  git push --set-upstream origin
```
3. Use IDE like Visual Studio to run application. This will bring Swagger up in the browser and allow you to test the service.
